# QR Code 扫码点餐 — 功能清单

> 每完成一项，将 `[ ]` 改为 `[x]`

---

## 一、后台管理端

### 1. 菜品管理
- [x] **菜品管理页面** — Compact 表格视图（菜名为主），可切换为顾客预览视图
- [x] **添加菜品** — 名称、价格、描述、分类、排序、上/下架
- [x] **编辑菜品** — 修改已有菜品信息
- [x] **删除菜品** — 删除菜品（软删除或硬删除）
- [x] **菜品子属性（规格选择）** — 每道菜可独立配置 N 组属性（如"辣度": 微辣/中辣/特辣），每个选项可选加价
- [ ] **菜品图片** — 上传/显示菜品图片
- [x] **菜品上下架** — 一键切换 available 状态
- [x] **服务员快捷修改** — 服务员可直接修改菜品信息（价格、上下架等）

### 2. 分类管理
- [x] **分类管理页面** — 查看、添加、编辑、删除分类
- [x] **分类排序** — 上下箭头手动调整分类顺序

### 3. 订单管理
- [x] **订单面板** — 查看所有订单，按状态筛选（全部/待处理/制作中/已完成）
- [x] **订单状态更新** — 待处理→制作中→已完成
- [x] **订单修改** — 服务员可修改已提交订单（增删菜品、修改规格/备注、重算价格）
- [x] **订单详情页** — 点击订单查看完整详情
- [x] **订单打印** — 打印小票

### 4. 桌台管理
- [x] **QR码生成页** — 为每张桌台生成二维码
- [x] **QR码打印** — 单张/全部打印
- [x] **桌台增删改** — 添加、编辑、删除桌台（含重名检查、占用保护）
- [x] **桌台状态概览** — 显示空闲/使用中状态 + 结算按钮

### 5. 门店设置
- [x] **门店信息编辑** — 店名、描述、营业时间（Logo 待图片上传服务）
- [x] **门店公告** — 设置顾客端显示的公告
- [x] **后台统一布局** — 左侧边栏导航 + 右侧内容区

---

## 二、顾客端（扫码后）

### 1. 扫码 & 菜单
- [x] **扫码着陆页** — 识别 storeId + tableId，跳转菜单
- [x] **菜单页** — 分类侧边栏 + 菜品列表 + 加购按钮
- [x] **菜品详情弹窗/页** — 查看大图、描述、选择规格（辣度/口味等子属性）
- [x] **菜品搜索** — 按关键词搜索菜品
- [ ] **菜品图片显示** — 菜单列表中展示图片

### 2. 购物车 & 下单
- [x] **购物车页** — 查看已选菜品、修改数量、备注
- [x] **提交订单** — 发送到后台
- [x] **下单成功页** — 显示订单号、桌号、菜品明细
- [x] **规格显示** — 购物车中显示已选规格（如"中辣 + 红烧"）
- [x] **加菜功能** — 已下单后可继续加菜（追加订单）

### 3. 订单查看
- [x] **我的订单页** — 购物车页显示当前桌台历史订单（只读）
- [x] **订单状态实时更新** — 轮询显示制作进度（10s 间隔）
