# Phase 5 Handoff 文档

本目录存放 Phase 5 协作的 live handoff 文档,**不是 freeze 归档**(目录名 archive 来自历史命名,语义上是 live)。

## 文件用途

- `phase-5-governance-digest.md` — 稳定治理规则,累积式追加 (规则/方法学/D 候选)
- `phase-5-state-snapshot.md` — 项目当前状态,**覆盖式重写**,每对话末由 Opus 产出最终版
- `phase-5-fabrication-archive.md` — Fabrication 历史原文,累积式追加 (供识别同类新 fabrication 锚点)

## 维护原则

- Digest 和 Archive 是累积的,只增不删
- Snapshot 是覆盖的,每对话末重写,杜绝 stale state 类问题 (#14)
- 历史 v1/v2/v3 转接包内容已蒸馏进本目录,原文在 Ian 本地 Downloads,不进 repo
